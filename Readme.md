## Overview

This project is an HCI (Human-Computer Interaction) Finals assignment that demonstrates web design principles through a modern, interactive website. The purpose is to showcase understanding of user interface design, user experience principles, and responsive web development. The key components include multi-page or single-page website architecture with parallax scrolling effects and transactional features for enhanced user engagement. The technology stack consists of HTML for semantic structure, CSS for styling and animations, and JavaScript for interactivity and dynamic behavior, all working together to create a cohesive, user-centered web experience.
